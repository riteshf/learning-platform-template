# AI & ML Essentials - 4 Month Course

## Course Overview

**Duration:** 4 months (16 weeks)  
**Schedule:** 3 classes per week (2 hours each)  
**Total Sessions:** 48 classes  
**Weekly Commitment:** ~6 hours live + 2-6 hours practice/self-work

---

## Course Structure

### Module 1: Data & Tools Fluency
**Duration:** 4 weeks (12 sessions)  
**Focus:** Python, SQL, Data Analysis, Data Visualization

### Module 2: Machine Learning Essentials
**Duration:** 6 weeks (18 sessions)  
**Focus:** ML Fundamentals, Supervised/Unsupervised Learning, Model Evaluation, Feature Engineering

### Module 3: Generative AI & AI Agents
**Duration:** 6 weeks (18 sessions)  
**Focus:** LLMs, Prompt Engineering, RAG, AI Agents, Production Deployment

---

## Daily Structure

Each class day includes:

- **Pre-class** (max 30 mins): Material to review, setup instructions, prerequisites
- **Live Session** (2 hours): Interactive instruction with hands-on coding
- **Post-class** (max 30 mins): Small assignment to reinforce learning

## Learning Outcomes

By the end of this course, you will be able to:

- Write Python code for data analysis and ML
- Query databases using SQL
- Build and evaluate machine learning models
- Work with Generative AI and LLMs
- Create AI agents and automation solutions
- Deploy ML models to production

---

## Assessment

- **Daily Assignments:** Small exercises after each class (30 mins)
- **Weekly Projects:** Larger projects at the end of each week
- **Module Projects:** Comprehensive projects at the end of each module
- **Capstone Project:** Final project combining all learnings

---

## Prerequisites

- Basic computer literacy
- Access to a computer with internet
- Willingness to learn and practice regularly

---

## Getting Started

1. Navigate to your current module folder (e.g., `module-1/`)
2. Review the module README for overview
3. Navigate to the current week folder (e.g., `module-1/week-01/`)
4. Review the week README for weekly goals
5. Navigate to the current day folder (e.g., `module-1/week-01/day-01/`)
6. Review the day README and follow: pre-class → live → post-class
