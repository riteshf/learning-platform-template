# Week 1: Python Fundamentals

## Overview

**Duration:** 3 sessions  
**Focus:** Building Python programming fundamentals

This week introduces you to Python programming. You'll learn the basics of Python syntax, control flow, functions, and data structures that form the foundation for all data science work.

---

## Learning Objectives

By the end of this week, you will be able to:

- Write Python code using variables, data types, and operators
- Use control flow statements (if/else, loops)
- Create and use functions
- Work with Python data structures (lists, dictionaries, tuples, sets)

---

## Daily Schedule

### Day 1: Python Basics & Environment Setup
- **Topics:** Variables, data types, operators, input/output
- **Assignment:** Simple calculator program

### Day 2: Control Flow & Functions
- **Topics:** Conditionals, loops, functions, scope
- **Assignment:** Number guessing game

### Day 3: Data Structures
- **Topics:** Lists, tuples, dictionaries, sets, list comprehensions
- **Assignment:** Student grade management system

---

## Weekly Project

Build a **Personal Budget Tracker** that:
- Accepts income and expense inputs
- Categorizes expenses
- Calculates monthly balance
- Displays summary statistics
