# Live Session: Day 1

**Duration:** 2 hours  
**Focus:** Hands-on practice with Python basics

---

## Session Checklist

### Part 1: Environment Setup (15 mins)
- [ ] Verify everyone has Python + VS Code working
- [ ] Demo: Run Python script vs interactive shell
- [ ] Show: How to read error messages

### Part 2: Variables & Types (25 mins)
- [ ] Demo: Creating variables with different types
- [ ] Demo: Using `type()` to check data types
- [ ] Demo: Type conversion with `int()`, `float()`, `str()`
- [ ] **Exercise 1:** Students create variables and print their types

### Part 3: Operators (25 mins)
- [ ] Demo: Arithmetic operators (`+`, `-`, `*`, `/`, `//`, `%`, `**`)
- [ ] Demo: Comparison operators (`==`, `!=`, `<`, `>`, `<=`, `>=`)
- [ ] Demo: Logical operators (`and`, `or`, `not`)
- [ ] **Exercise 2:** Students practice operators with given values

### Part 4: Input/Output (25 mins)
- [ ] Demo: Using `print()` with multiple values
- [ ] Demo: Using `input()` to get user input
- [ ] Demo: Converting input string to int/float
- [ ] **Exercise 3:** Students create interactive greeting program

### Part 5: Putting It Together (25 mins)
- [ ] **Exercise 4:** Build simple calculator (scaffold for homework)
- [ ] Students work: Get two numbers and operator, perform calculation
- [ ] Discuss: Error handling (division by zero, invalid input)

### Part 6: Wrap-up (5 mins)
- [ ] Review key concepts covered
- [ ] Explain post-class assignment
- [ ] Q&A
