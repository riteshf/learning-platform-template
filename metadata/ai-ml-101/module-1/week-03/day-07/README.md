# Day 7: Data Manipulation

## Overview

**Duration:** 2 hours live + 30 mins pre-class + 30 mins post-class  
**Focus:** Advanced data manipulation with Pandas

Today you'll learn advanced techniques for manipulating data in Pandas, including filtering, sorting, grouping, and merging operations.

---

## Pre-class (30 mins)

### Materials to Review
- Review Day 6 concepts (Pandas basics)
- [Pandas Data Manipulation](https://pandas.pydata.org/docs/user_guide/basics.html)
- [GroupBy Operations](https://pandas.pydata.org/docs/user_guide/groupby.html)

### Prerequisites
- Completed Day 6 assignments
- Understanding of DataFrames

---

## Live Session (2 hours)

### Topics Covered

1. **Filtering Data**
   - Boolean indexing
   - Query method
   - Filtering with conditions

2. **Sorting Data**
   - Sorting by columns
   - Multi-column sorting
   - Sorting by index

3. **Grouping Operations**
   - GroupBy basics
   - Aggregation functions
   - GroupBy with multiple columns
   - Transform and apply functions

4. **Merging DataFrames**
   - Inner, outer, left, right joins
   - Merge on multiple columns
   - Concatenating DataFrames

### Hands-on Exercises
- Filtering and sorting exercises
- GroupBy operations
- Merging multiple datasets
- Complex data manipulation scenarios

---

## Post-class (30 mins)

### Assignment: Sales Data Analysis Project

Create a Python program that:
1. Loads a sales dataset
2. Filters data by date range and product category
3. Groups data by region and calculates totals
4. Merges with customer data
5. Generates summary statistics
6. Identifies top-selling products

**Requirements:**
- Use advanced filtering and sorting
- Perform GroupBy operations
- Merge multiple DataFrames
- Generate meaningful insights

**Submission:**
- Save your code as `sales_analysis.py`
- Include the dataset you used
- Be ready to present your findings
