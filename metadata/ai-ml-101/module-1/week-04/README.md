# Week 4: SQL Fundamentals

## Overview

**Duration:** 3 sessions  
**Focus:** SQL database querying and Python-SQL integration

This week introduces you to SQL, the language for working with relational databases. You'll learn to write queries and integrate SQL with Python.

---

## Learning Objectives

By the end of this week, you will be able to:

- Write SQL queries to extract data
- Use JOINs to combine data from multiple tables
- Perform aggregations and grouping
- Integrate SQL with Python
- Build data pipelines using SQL and Python

---

## Daily Schedule

### Day 10: SQL Basics
- **Topics:** SELECT, WHERE, ORDER BY, LIMIT
- **Assignment:** Query practice exercises

### Day 11: SQL Advanced Queries
- **Topics:** JOINs, GROUP BY, HAVING, subqueries
- **Assignment:** Complex query challenges

### Day 12: SQL with Python
- **Topics:** SQLite/PostgreSQL with Python, pandas + SQL
- **Assignment:** Build a data pipeline project

---

## Weekly Project

Build a **Database Analysis System** that:
- Connects to a database
- Performs complex queries
- Integrates with Pandas for analysis
- Creates visualizations from query results
- Generates automated reports
