# Day 10: SQL Basics

## Overview

**Duration:** 2 hours live + 30 mins pre-class + 30 mins post-class  
**Focus:** Introduction to SQL and basic querying

Today you'll learn the fundamentals of SQL, including how to write basic queries to retrieve and filter data from databases.

---

## Pre-class (30 mins)

### Materials to Review
- [SQL Introduction](https://www.w3schools.com/sql/sql_intro.asp)
- [SQL SELECT Statement](https://www.w3schools.com/sql/sql_select.asp)
- Database concepts (tables, rows, columns)

### Setup Instructions
1. SQLite comes with Python (no installation needed)
2. Download a sample database or create one
3. Install a SQL client (optional): DB Browser for SQLite

### Prerequisites
- Completed Week 3 assignments
- Understanding of data structures

---

## Live Session (2 hours)

### Topics Covered

1. **SQL Introduction**
   - What is SQL?
   - Relational databases
   - Tables, rows, and columns

2. **SELECT Statement**
   - Basic SELECT
   - Selecting specific columns
   - SELECT DISTINCT

3. **WHERE Clause**
   - Filtering rows
   - Comparison operators
   - Logical operators (AND, OR, NOT)
   - IN, BETWEEN, LIKE operators

4. **ORDER BY and LIMIT**
   - Sorting results
   - Ascending and descending order
   - Limiting results

### Hands-on Exercises
- Writing basic SELECT queries
- Filtering data with WHERE
- Sorting and limiting results
- Combining multiple conditions

---

## Post-class (30 mins)

### Assignment: Query Practice Exercises

Create SQL queries for the following tasks:
1. Select all columns from a table
2. Select specific columns with aliases
3. Filter rows using WHERE with multiple conditions
4. Sort results by multiple columns
5. Limit results to top N records
6. Use LIKE to search for patterns
7. Use BETWEEN for range queries

**Requirements:**
- Write queries for a provided database
- Test all queries
- Document what each query does
- Handle edge cases

**Submission:**
- Save your queries as `sql_practice.sql`
- Include query results
- Be ready to explain your queries

---

## Learning Outcomes

After completing this day, you should be able to:
- ✅ Write basic SELECT queries
- ✅ Filter data using WHERE clause
- ✅ Sort and limit query results
- ✅ Use various SQL operators
- ✅ Understand database structure
