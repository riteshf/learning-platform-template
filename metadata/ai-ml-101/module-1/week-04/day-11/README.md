# Day 11: SQL Advanced Queries

## Overview

**Duration:** 2 hours live + 30 mins pre-class + 30 mins post-class  
**Focus:** Advanced SQL techniques including JOINs, aggregations, and subqueries

Today you'll learn advanced SQL techniques to combine data from multiple tables and perform complex aggregations.

---

## Pre-class (30 mins)

### Materials to Review
- Review Day 10 concepts (SQL basics)
- [SQL JOINs](https://www.w3schools.com/sql/sql_join.asp)
- [SQL GROUP BY](https://www.w3schools.com/sql/sql_groupby.asp)

### Prerequisites
- Completed Day 10 assignments
- Understanding of basic SQL queries

---

## Live Session (2 hours)

### Topics Covered

1. **JOINs**
   - INNER JOIN
   - LEFT JOIN / RIGHT JOIN
   - FULL OUTER JOIN
   - Self JOINs
   - Multiple table JOINs

2. **Aggregations**
   - COUNT, SUM, AVG, MIN, MAX
   - GROUP BY clause
   - HAVING clause
   - Aggregate functions with WHERE

3. **Subqueries**
   - Subqueries in WHERE clause
   - Subqueries in SELECT clause
   - Correlated subqueries
   - EXISTS and IN with subqueries

4. **Advanced Techniques**
   - CASE statements
   - Window functions (intro)
   - UNION and UNION ALL

### Hands-on Exercises
- JOINing multiple tables
- Grouping and aggregating data
- Writing subqueries
- Complex query scenarios

---

## Post-class (30 mins)

### Assignment: Complex Query Challenges

Create SQL queries for the following challenges:
1. JOIN multiple tables to get related data
2. Calculate aggregations grouped by categories
3. Use HAVING to filter aggregated results
4. Write a subquery to find top performers
5. Use CASE statements for conditional logic
6. Combine multiple queries with UNION

**Requirements:**
- Write efficient, readable queries
- Use appropriate JOIN types
- Handle NULL values
- Optimize query performance

**Submission:**
- Save your queries as `advanced_queries.sql`
- Include query results and explanations
- Be ready to present your solutions

- ✅ Write subqueries
- ✅ Use HAVING for filtered aggregations
- ✅ Combine multiple SQL techniques
