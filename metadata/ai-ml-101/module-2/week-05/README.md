# Week 5: ML Fundamentals

## Overview

**Duration:** 3 sessions  
**Focus:** Introduction to machine learning and data preprocessing

This week introduces you to machine learning concepts and prepares you to work with ML libraries. You'll learn about different types of learning, data preprocessing techniques, and the scikit-learn framework.

---

## Learning Objectives

By the end of this week, you will be able to:

- Understand machine learning concepts and types
- Preprocess data for machine learning
- Use scikit-learn for ML workflows
- Build your first ML pipeline

---

## Daily Schedule

### Day 13: Introduction to Machine Learning
- **Topics:** ML overview, supervised vs unsupervised learning, use cases
- **Assignment:** Identify ML use cases in daily life

### Day 14: Data Preprocessing for ML
- **Topics:** Train/test split, scaling, encoding, feature selection
- **Assignment:** Preprocess a dataset for ML

### Day 15: Scikit-learn Basics
- **Topics:** Scikit-learn API, pipeline, cross-validation
- **Assignment:** Build your first ML pipeline

---

## Weekly Project

Preprocess and prepare a real-world dataset for machine learning:
- Load and explore the data
- Handle missing values and outliers
- Encode categorical variables
- Scale numerical features
- Split into train/test sets

---

## Prerequisites

- Completion of Module 1
- Understanding of Pandas and NumPy
- Basic statistics knowledge
