# Week 6: Supervised Learning - Regression

## Overview

**Duration:** 3 sessions  
**Focus:** Regression algorithms for predicting continuous values

This week focuses on regression algorithms, starting with linear regression and moving to more advanced techniques like polynomial regression and regularization.

---

## Learning Objectives

By the end of this week, you will be able to:

- Understand regression problems
- Build linear regression models
- Handle non-linear relationships
- Apply regularization techniques
- Evaluate regression models

---

## Daily Schedule

### Day 16: Linear Regression
- **Topics:** Simple & multiple linear regression, evaluation metrics
- **Assignment:** Predict house prices

### Day 17: Polynomial & Regularization
- **Topics:** Polynomial regression, Ridge, Lasso
- **Assignment:** Regularization practice

### Day 18: Regression Project
- **Topics:** End-to-end regression project
- **Assignment:** Complete and submit project

## Weekly Project

Build a **Complete Regression Model** that:
- Predicts a continuous target variable
- Handles multiple features
- Applies appropriate preprocessing
- Uses regularization if needed
- Evaluates model performance comprehensively
- Generates predictions and insights
