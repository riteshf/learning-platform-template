# Week 7: Supervised Learning - Classification

## Overview

**Duration:** 3 sessions  
**Focus:** Classification algorithms for predicting categorical values

This week focuses on classification algorithms, including logistic regression, tree-based methods, and other classification techniques.

---

## Learning Objectives

By the end of this week, you will be able to:

- Understand classification problems
- Build various classification models
- Evaluate classification models
- Compare different classifiers
- Choose appropriate algorithms

---

## Daily Schedule

### Day 19: Logistic Regression
- **Topics:** Logistic regression, decision boundaries, metrics
- **Assignment:** Binary classification exercise

### Day 20: Decision Trees & Random Forest
- **Topics:** Decision trees, random forest, feature importance
- **Assignment:** Classification project with trees

### Day 21: SVM & KNN
- **Topics:** Support Vector Machines, K-Nearest Neighbors
- **Assignment:** Compare different classifiers

---

## Weekly Project

Build a **Multi-class Classification Model** that:
- Handles multiple classes
- Uses appropriate preprocessing
- Compares multiple algorithms
- Evaluates comprehensively
- Generates predictions and insights

---

## Prerequisites

- Completion of Week 6
- Understanding of regression concepts
- Knowledge of evaluation metrics
