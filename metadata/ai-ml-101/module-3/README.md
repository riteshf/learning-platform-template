# Module 3: Generative AI & AI Agents

## Overview

**Duration:** 6 weeks (18 sessions)  
**Focus:** Large Language Models, Prompt Engineering, RAG, AI Agents, and Production Deployment

This module covers the latest developments in Generative AI. You'll learn to work with LLMs, build RAG systems, create AI agents, and deploy AI applications to production.

---

## Learning Objectives

By the end of this module, you will be able to:

- Work with Large Language Models (LLMs) and APIs
- Write effective prompts and optimize LLM outputs
- Build Retrieval Augmented Generation (RAG) systems
- Create AI agents with tool integration
- Deploy and monitor AI applications in production

---

## Weekly Breakdown

### Week 11: Introduction to Generative AI
- AI landscape, LLMs, prompt engineering basics, OpenAI API

### Week 12: Advanced Prompt Engineering
- Advanced prompting techniques, templates, LLM evaluation

### Week 13: RAG (Retrieval Augmented Generation)
- RAG architecture, vector databases, building RAG systems

### Week 14: AI Agents
- Agent architecture, building agents, multi-agent systems

### Week 15: Production & Deployment
- Model deployment, LLM app deployment, monitoring

### Week 16: Capstone Project
- End-to-end AI/ML solution development and presentation

---

## Module Project

Build a production-ready AI application that includes:
- LLM integration with custom prompts
- RAG system for knowledge retrieval
- AI agent with tool integration
- Deployment and monitoring

---

## Prerequisites

- Completion of Module 1 (Python fundamentals)
- Completion of Module 2 (Machine Learning basics)
