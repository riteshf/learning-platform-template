# Week 11: Introduction to Generative AI

## Overview

**Duration:** 3 sessions  
**Focus:** Introduction to Large Language Models and prompt engineering

This week introduces you to the world of Generative AI and Large Language Models. You'll learn about the AI landscape, transformer architecture, and how to work with LLM APIs.

---

## Learning Objectives

By the end of this week, you will be able to:

- Understand the AI landscape and LLM architecture
- Write effective prompts for LLMs
- Use OpenAI API to interact with GPT models
- Build simple LLM-powered applications

---
