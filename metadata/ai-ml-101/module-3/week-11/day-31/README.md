# Day 31: AI Landscape & LLMs

## Overview

**Duration:** 2 hours live + 30 mins pre-class + 30 mins post-class  
**Focus:** Introduction to Generative AI and Large Language Models

Today you'll learn about the AI landscape, the evolution of AI, transformer architecture, and an overview of Large Language Models.

---

## Live Session (2 hours)

### Topics Covered

1. **AI Landscape**
   - Evolution of AI
   - Traditional AI vs Generative AI
   - Current AI capabilities
   - Future trends

2. **Transformer Architecture**
   - Attention mechanism
   - Encoder-decoder architecture
   - Self-attention
   - Positional encoding

3. **Large Language Models (LLMs)**
   - What are LLMs?
   - How LLMs work
   - Popular LLMs (GPT, Claude, Llama)
   - LLM capabilities and limitations

4. **LLM Applications**
   - Text generation
   - Question answering
   - Summarization
   - Translation
   - Code generation

### Hands-on Exercises
- Exploring LLM capabilities
- Understanding transformer concepts
- Comparing different LLMs
- Identifying use cases

## Post-class (30 mins)

### Assignment: Research Different LLM Models

Create a research document that:
1. Compares at least 3 different LLM models
2. Describes each model's architecture
3. Lists capabilities and limitations
4. Identifies best use cases for each
5. Discusses cost and accessibility

**Requirements:**
- Research GPT-4, Claude, and one open-source model
- Compare features and capabilities
- Consider practical applications
- Document your findings

**Submission:**
- Save as `llm_research.md`
- Be ready to discuss your findings
