# Day 32: Prompt Engineering Basics

## Overview

**Duration:** 2 hours live + 30 mins pre-class + 30 mins post-class  
**Focus:** Writing effective prompts for LLMs

Today you'll learn the fundamentals of prompt engineering - the art and science of crafting prompts to get the best results from LLMs.

---

## Pre-class (30 mins)

### Materials to Review
- Review Day 31 concepts (LLMs)
- [Prompt Engineering Guide](https://www.promptingguide.ai/)
- [OpenAI Prompt Engineering](https://platform.openai.com/docs/guides/prompt-engineering)

### Prerequisites
- Completed Day 31 assignments
- Understanding of LLMs

---

## Live Session (2 hours)

### Topics Covered

1. **What is Prompt Engineering?**
   - Definition and importance
   - Why prompts matter
   - Prompt vs query

2. **Writing Effective Prompts**
   - Clear instructions
   - Context provision
   - Output format specification
   - Examples and templates

3. **Prompting Techniques**
   - Zero-shot prompting
   - Few-shot learning
   - Chain-of-thought prompting
   - Role-based prompting

4. **Best Practices**
   - Being specific
   - Providing context
   - Using examples
   - Iterative refinement

### Hands-on Exercises
- Writing basic prompts
- Improving prompts iteratively
- Using few-shot examples
- Chain-of-thought exercises

---

## Post-class (30 mins)

### Assignment: Prompt Engineering Exercises

Create prompts for the following tasks:
1. Text summarization
2. Question answering
3. Code generation
4. Data extraction
5. Creative writing
6. Analysis task

For each task:
- Write initial prompt
- Refine based on results
- Document what works
- Explain your approach

**Requirements:**
- Create at least 6 different prompts
- Test and refine each
- Document your process
- Explain improvements

**Submission:**
- Save as `prompt_exercises.md`
- Include prompts and results
- Be ready to share your best prompts
