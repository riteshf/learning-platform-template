# Day 33: Working with OpenAI API

## Overview

**Duration:** 2 hours live + 30 mins pre-class + 30 mins post-class  
**Focus:** Using OpenAI API to interact with GPT models

Today you'll learn to use the OpenAI API to build applications with GPT models, including API setup, parameters, and cost management.

---

## Pre-class (30 mins)

### Materials to Review
- Review Day 32 concepts (prompt engineering)
- [OpenAI API Documentation](https://platform.openai.com/docs/api-reference)
- [OpenAI Python Library](https://github.com/openai/openai-python)

### Setup Instructions
1. Create OpenAI account
2. Get API key
3. Install: `pip install openai`
4. Set up API key securely

### Prerequisites
- Completed Day 32 assignments
- Understanding of prompts
- OpenAI API access

---

## Live Session (2 hours)

### Topics Covered

1. **OpenAI API Setup**
   - Creating account
   - Getting API key
   - Setting up authentication
   - Security best practices

2. **API Basics**
   - Making API calls
   - Chat completions endpoint
   - Request structure
   - Response handling

3. **Model Parameters**
   - Temperature
   - Max tokens
   - Top-p
   - Frequency and presence penalties
   - Stop sequences

4. **Cost Management**
   - Understanding pricing
   - Token counting
   - Cost optimization
   - Rate limits

### Hands-on Exercises
- Making API calls
- Adjusting parameters
- Handling responses
- Building simple applications

---

## Post-class (30 mins)

### Assignment: Build a Simple Chatbot

Create a Python program that:
1. Uses OpenAI API
2. Implements a simple chatbot
3. Handles conversation context
4. Allows user interaction
5. Manages API errors gracefully
6. Includes cost tracking

**Requirements:**
- Use OpenAI Python library
- Implement conversation flow
- Handle errors appropriately
- Track token usage
- Create user-friendly interface

**Submission:**
- Save your code as `simple_chatbot.py`
- Include error handling
- Be ready to demonstrate your chatbot
