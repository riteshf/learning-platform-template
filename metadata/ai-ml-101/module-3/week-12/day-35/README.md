## Post-class (30 mins)

### Assignment: Build a Prompt Library

Create a prompt library system that:
1. Stores reusable prompt templates
2. Supports variable substitution
3. Organizes prompts by category
4. Includes documentation
5. Provides examples
6. Allows versioning

**Requirements:**
- Create at least 10 prompt templates
- Organize by use case
- Include documentation
- Provide usage examples
- Make it reusable

**Submission:**
- Save as `prompt_library.py` or organized files
- Include documentation
- Be ready to share your library
